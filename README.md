# Qortex Labs

**AI-Powered Quality Engineering Tools**

A portfolio of self-hosted, AI-native tools built from scratch for modern QA teams. No vendor lock-in. No templates. Every line intentional.

## The Products

### 🔥 API Qortex
AI-powered API testing and documentation platform.

- 4 API protocols (REST, GraphQL, gRPC, WebSocket)
- 27 visual assertions with diff highlighting
- 6-provider AI fallback chain
- Built-in security scanner and load tester
- 9 authentication methods, 4-role RBAC

**Stack:** Next.js 15 · TypeScript · Prisma · SQLite · Tailwind · Zustand

🔗 [Showcase](https://krishpravn.github.io/qortex-labs/api-qortex/) · [Repository](https://github.com/krishpravn/api-qortex)

---

### 🔷 Phoenix TestAI
AI-native test management and execution platform.

- 6 autonomous AI agents (Triage, Discovery, Strategy, Learning, Generator, Executor)
- MCP integration (Playwright, Filesystem, GitHub)
- 26-tool AI chat with context-aware assistance
- Playwright test execution with live SSE streaming
- 11 testing types, 8 report types, 4-theme design system, 5-role RBAC

**Stack:** Next.js 15 · React 19 · TypeScript · Prisma · Playwright · MCP · Zustand

🔗 [Showcase](https://krishpravn.github.io/qortex-labs/phoenix-testai/)

## Engineering Philosophy

- **Self-Hosted First** — Your data stays on your infrastructure. Zero cloud dependency.
- **AI-Native, Not AI-Bolted** — Multi-provider routing, cost-optimized inference, autonomous agents.
- **Production Grade** — RBAC, audit trails, event-driven architecture, SSE streaming.
- **No Templates** — Every product built from the ground up. No boilerplate generators.
- **Zero Vendor Lock-in** — No SaaS subscriptions, no third-party data processors.
